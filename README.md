## Run server

1. Go to the project directory

1. Install all dependencies

   ```shell
   npm install
   ```

1. Run server

   ```shell
   node build/src/server
   ```

1. Run client
   ```shell
   node build/src/client
   ```
1. Open two terminals for clients (the port is running on 127.0.0.1:5050)
